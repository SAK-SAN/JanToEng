# JanToEng - 日⇒英 英訳ツール

簡単な日本語を入力するだけで、国際会議や学術的なシチュエーションで利用可能なレベルの英訳を生成するStreamlitアプリケーションです。GoogleのGemini APIを利用し、ターゲット層や用途に合わせた3段階の翻訳レベル（中学生レベル、高校生レベル、ネイティブ・論文レベル）を選択して最適な英文を構築できます。

## 🚀 特徴

* **Geminiモデルの動的取得**: 設定されたAPIキーを使って、テキスト生成（`generateContent`）に対応している最新の利用可能なモデルを自動で取得・選択できます。
* **3段階の翻訳レベル調整**:
  * **中学生レベル**: 複雑な内容を簡潔な語彙（英検3級〜準2級程度）と短い一文に書き換え、直感的に伝わる構成にします。
  * **高校生レベル**: 論理の展開を明確にする接続詞や標準的な構文を用い、論理的かつスムーズに伝わる文章にします。
  * **ネイティブ・論文レベル**: トップジャーナルや主要学会で通用する、格調高く自然なアカデミック英語（洗練された学術的語彙や名詞化表現）に変換します。
* **翻訳ポイントの解説機能**: 単なる翻訳結果の出力だけでなく、なぜその語彙や構文を選択したのかの解説テキストも同時に表示されます。
* **ワンクリックコピー**: Streamlitのコードブロック機能により、生成された英文をワンクリックで簡単にクリップボードへコピーできます。
## 🛠️ 技術スタック

* **Language**: Python
* **Framework**: Streamlit
* **HTTP Client**: requests（Gemini APIとの通信用）

## 📦 セットアップと実行方法

### 1. リポジトリのクローン
ローカル環境にリポジトリをクローンし、ディレクトリに移動します。

```bash
git clone [https://github.com/SAK-SAN/JanToEng.git](https://github.com/SAK-SAN/JanToEng.git)
cd JanToEng
```

### 2. 依存関係のインストール
必要なライブラリをインストールします。

```bash
pip install -r requirements.txt
```
### 3. 環境変数の設定
プロジェクトのルートディレクトリに `.env` ファイルを作成し、ご自身のGemini APIキーを設定してください。
※ `.env` ファイルは `.gitignore` に登録されているため、GitHub等に誤って公開される心配はありません。

```text
GEMINI_API_KEY=your_gemini_api_key_here
```

### 4. アプリケーションの起動
Streamlitサーバーを起動します。ブラウザが自動的に立ち上がり、アプリの画面が表示されます。

```bash
streamlit run app.py
```

## 🌐 クラウドへの展開（Streamlit Cloudなど）

Streamlit Cloudなどのホスティングサービスにデプロイする場合は、各種プラットフォームの環境変数設定（Secrets機能など）に以下のキーを登録してください。

* **Key**: `GEMINI_API_KEY`
* **Value**: *（あなたのGemini APIキー）*
